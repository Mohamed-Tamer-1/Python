Developed a web scraper using Selenium to automate the extraction of match data from YallaKora. Key data extracted included:

Team Names: Both teams participating in matches.
Match Scores: Real-time scores and statistics.
Match Timings: Scheduled times for each match.
Channels: Broadcasting channels for live matches.
Match Stages: Stages of the matches.
The data was organized and exported in CSV and JSON formats for analysis.
